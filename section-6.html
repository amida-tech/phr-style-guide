Style Guide

6.0 - Colors

The colors to be used on the site. These colors should generally inherit to the rest of the site. Grayscale elements aren't defined in this area; instead they are referenced by component.

6.1 - Color Scheme

The site's overall color scheme should create a calming experience to clearly convey health information without creating undue sensations of alarm or excitement. A monochromatic color scheme of a subtle blue shades serves this purpose well. By having a muted basic color scheme, we can better accent critical components of health data.

Default styling

Lighter accent color for less visual weight.

.light-blue

Deeper accent color for heavier visual weight.

.deep-blue

6.2 - Contrasting Colors

Several contrasting colors can be used to highlight important information; for instance stress alerts
about dangerous test results, or warn about changes or action which might be required. These colors are complementary to the base colors to drive contrast, and should be used very sparingly. These colors should be presented contrasting with the base colors.

Default styling

Muted Orange for standard colors/general alerts.

.basic-orange

Intense Orange color for extreme contrast, should be used very sparingly.

.dark-orange
