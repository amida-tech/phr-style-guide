3.0 - Text Styling

This style guide uses open fonts freely available through Google Fonts. This guide uses 'Open Sans,' 'Montserrat,' and 'Roboto Condensed.'

3.1.1 - Heading 1

Heading level one should be used sparingly, and only to communicate very important information.

Default styling

You know nothing, Jon Snow.

 

<div class="text-center">
  <h1>You know nothing, Jon Snow.</h1>
</div>

3.1.2 - Heading 2

Level two headers should act as sub-headers; for instance this header also looks great offset a row below level one headers.

Default styling

Winter is coming.

 

<div class="text-center">
  <h2>Winter is coming.</h2>
</div>

3.1.3 - Heading 3

Heading level three is useful as a header of sub-elements on pages, for instance a block of information regarding a specific allergy.

Default styling

There is only one thing we say to death, "Not today!"

 

<div class="text-center">
  <h3>There is only one thing we say to death, "Not today!"</h3>
</div>

3.1.4 - Heading 4

Level four headings carry roughly equivalent visual weight with standard text. It is useful for providing variety in the expression of dense areas of text.

Default styling

A Lannister always pays his debts.

 

<div class="text-center">
  <h4>A Lannister always pays his debts.</h4>
</div>

3.1.5 - Heading 5

Level five headings should be used for communication of sub-primary page information, for instance breadcrumbs and picture captions.

Default styling

I learned how to die a long time ago.

 

<div class="text-center">
  <h5>I learned how to die a long time ago.</h5>
</div>

3.1.6 - Heading 6

Heading level six should be used extremely rarely, for information such as links to annotation or copyright information.

Default styling

Wise men do not make demands of kings.

 

<div class="text-center">
  <h6>Wise men do not make demands of kings.</h6>
</div>

3.2.1 - Coded Text

Coded text can be used to express encoded medical data or data which may appear on a printed label, for instance a medication SIG or an ICD encoded entry. Generally, they should appear during drill-down into a record's sub-elements.

Default styling

Take 2 pills 3x daily

 

<code>Take 2 pills 3x daily</code>
