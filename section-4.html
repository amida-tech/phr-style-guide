4.0 - Navigation

Covers structure and styling of navigation elements throughout the site.

4.1 - Primary Navigation

A primary navigaton bar across the top of the page provides a static placeholder for information, while freeing up the body of the site for the display of content specific information.

In applications which have multiple users, and in which a user may be viewing records of various individuals, it is important to differentiate between the current user and the record of the individual. A common design practice is to embed the current user's information in the navigation bar. This provides user-level context and will allow a system user to visually differentiate between themselves, and the individual associated with any given record.

If search is a feature of the site, it should be incorporated into the navigation bar. This will allow easy access from any area of the site.

4.2 - Secondary Navigation

The primary navigation of the site should be statically fixed to the top left of the page. The navigation should feature a user picture and name, so at any time there can be certainty as to who's record is being viewed.
